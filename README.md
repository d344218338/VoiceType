# VoiceType - 免费本地AI语音助手

对着电脑说话，AI自动帮你整理成书面文字、翻译、改写，直接输入到当前窗口。

**完全免费，全部本地运行，不上传任何数据。**

## 三大功能

| 功能 | 说明 | 默认快捷键 |
|------|------|-----------|
| 语音整理 | 说话 → AI整理成通顺的书面文字 → 自动输入 | 右Alt |
| 语音翻译 | 说话 → 自动翻译成目标语言 → 自动输入 | 右Alt + 右Shift |
| AI改写 | 选中文字 → 说指令（如"润色"） → AI改写替换 | 右Alt + 右Ctrl |

## 一键安装（Windows）

1. 下载本项目，解压
2. 双击 `VoiceType-一键安装.bat`
3. 等待自动安装完成（约5分钟）
4. 双击桌面「VoiceType 语音助手」图标即可使用

安装程序会自动帮你装好 Python、Ollama、AI模型，全程无需手动操作。

## 使用方法

1. 启动 VoiceType（后台运行，右下角托盘图标）
2. 在任何地方按一下快捷键 → 开始录音
3. 对着麦克风说话
4. 再按一下同样的快捷键 → 停止录音
5. 等几秒，结果自动输入到当前窗口

## 终端模式

```
vt transcribe    # 语音整理
vt translate     # 语音翻译
vt chat          # 跟AI聊天
```

## 技术栈

- 语音识别：Google Speech Recognition（在线快速） + faster-whisper（离线回退）
- AI模型：Ollama + qwen2.5:1.5b（本地运行，约1GB）
- 界面：PyQt6 深色主题
- 全局快捷键：pynput（支持左右修饰键区分）

## 系统要求

- Windows 10/11
- 麦克风
- 建议 8GB 以上内存
